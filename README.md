# CareXpert
CareXpert is an Android-based health application that provides complete health solutions and a trusted community to meet the health needs of you and your family. This project is intended for the Android Based Application Programming course final project at Petra Christian University.

### Home Activity Menu
Users can choose one of the entire list of provinces in Indonesia as a filter from the list of all posts. After selecting a province, the user will also be given a list of districts/cities in the province that has been selected. The list of provinces and districts/cities is obtained from the Indonesian Regional Data API (https://github.com/emsifa/api-region-indonesia).<br><br>
